What is CPU Scheduling?

CPU scheduling is a crucial aspect of operating systems that deals with the efficient allocation of the CPU (Central Processing Unit) to processes in a computer system. This procedure makes maximum use of the CPU by allowing one process to use it while another is delayed (on standby) due to the unavailability of any resources, such as I/O, etc. CPU scheduling is used to increase the system's speed, fairness, and efficiency. The operating system has to choose one of the processes in the queue that is available in the ready-to-use line whenever the CPU becomes idle. A temporary CPU scheduler is responsible for the selection procedure. The Scheduler allocates the CPU to one of the memory processes that are prepared for launch.

There are two types of scheduling methods and these are:

Preemptive Scheduling: It is used when a process switches from running state to ready state or from the waiting state to the ready state.

Non-Preemptive Scheduling: It is used when a process terminates , or when a process switches from running state to waiting state.
